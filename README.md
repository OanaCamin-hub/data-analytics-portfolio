# Data Analytics Portfolio – Oana Camin

## Project 1: Restaurant Operations Analysis
Analyzed 30 days of operational data to identify inefficiencies in food cost, labor, and waste.

### Analysis Performed
- Calculated Food Cost %, Labor %, and Average Order Value
- Compared weekday vs weekend performance
- Analyzed trends in cost and waste

### Key Insights
- Higher food cost and waste during peak periods
- Waste increases during high sales days
- Labor inefficiencies during slower periods

### Tools Used
- Google Sheets (Excel)

## Project 2: Customer Behavior & Sales Analysis
Used SQL to analyze transaction data and identify revenue drivers.

### Analysis Performed
- Aggregated data using GROUP BY
- Used CASE statements to compare weekend vs weekday performance
- Calculated revenue contribution by category

### Key Insights
- Bowl category generates the highest revenue
- Weekend sales are higher than weekdays
- Higher item count increases order value

### Tools Used
- SQL
